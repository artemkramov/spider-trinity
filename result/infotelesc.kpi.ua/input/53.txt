GROUNDS</h3></div><div><em>Leonid Uryvskyi, Kateryna Prokopenko, Anton Peshkin</em></div><br><div><h4>Abstract</h4><br><div><p>The convolutional codes analysis and synthesis technique is proposed. This technique is optimal according&nbsp;to near Shannon's bound criterion and guarantees a given bit error rate from message source. The proposed&nbsp;technique is based on the well-known optimum block codes synthesis method. As a result, equivalent parameters&nbsp;set of convolutional codes is received. This set corresponds the optimal block code. The algorithm for synthesis an&nbsp;optimal convolutional code is proposed. Examples of optimal convolutional codes analysis and synthesis in the&nbsp;channel with a given energy are presented.</p></div><br></div><div><h4>Full Text:</h4><a href="http://infotelesc.kpi.ua/article/view/35930/32211">PDF</a></div><div><h4>References</h4><br><div><p>C.
