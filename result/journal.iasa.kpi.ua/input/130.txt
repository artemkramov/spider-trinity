Using Modified Morphological Analysis Method</h3></div><div><em>I. O. Savchenko</em></div><br><div><h4>Abstract</h4><br><div>Methods of taking into an account the time parameter in modified morphological analysis problems are considered in this paper. A quasi-static problem definition is considered, which includes such time dependent factors as events and tendencies. A method of taking these factors into account within a morphological table is given and its flaws are stated. A newly developed technique for including events and tendencies as external elements for the morphological table, that introduce scale coefficients into the modified morphological analysis problem, is presented. A solution for the problem of object’s evolution considering its changes through time, is shown. The issue of supporting a morphological table is considered and methods are presented to guarantee this morphological table to be up-to-date during time periods.</div><br></div><div><h4>Full Text:</h4><a href="http://journal.iasa.kpi.ua/article/view/52088/47960">PDF (Русский)</a></div><div><h4>References</h4><br><div><p>Zgurovskiy M.Z., Pankratova N.D.
