medical records</h3></div><div><em>Anatoly Petrenko, Roman Kyslyi, Ihor Pysmennyi</em></div><br><div><h4>Abstract</h4><br><div>Storing and sharing healthcare data is challenging. Despite different data types that can be used on different platforms, also there is a question of security of gathered data. Storing data in a traditional way may cause data leak and it unavailability in critical moments. In this work we present decentralized way of storing patient data that can be used to avoid security and unavailability problems. A blockchain is a distributed and decentralized ledger that contains connected blocks of transactions. Unlike other ledger approaches, blockchain guarantees tamper proof storage of approved transactions.</div><br></div><div><h4>Keywords</h4><br><div>Cloud computing; Medical records; Distributed databases; Computer science; Online carehelping; Sensor data storage</div><br></div><div><h4>Full Text:</h4><a href="http://journal.iasa.kpi.ua/article/view/117178/111276">PDF</a></div><div><h4>GOST Style Citations</h4><br><div>—</div></div><br><br><br><br>DOI:<a href="https://doi.org/10.20535/SRIT.2308-8893.2017.3.01">https://doi.org/10.20535/SRIT.2308-8893.2017.3.01</a><div></div><h3>Refbacks</h3><ul><li>There are currently no refbacks.</li></ul></div></div></div></div>
