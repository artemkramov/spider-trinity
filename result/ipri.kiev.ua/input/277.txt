The formulae for the calculation of speckle contrast, which can be applied to any scanning laser projector, are obtained. It is shown that the main requirement for successful speckle suppression in a scanning laser projector is a narrow autocorrelation peak and low side-lobes level in the autocorrelation function of the complex amplitude distribution across a scanning light beam. It is shown that the application of diffraction optical elements with a Barker code phase shape can use only natural laser projector scanning motion for speckle suppression. Fig.: 7.
